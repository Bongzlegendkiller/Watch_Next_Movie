# INSTALLING AND RUNNING Watch_Next PYTHON SCRIPTS
--Using Commande Prompt--

Step 1
-------
Start Docker Desktop

```bash
docker -d
```


Step 2
-------
Build Docker image

```bash
docker build -t watch_next ./
```

Step 3
-------
Run Docker Image

```bash
docker run -it watch_next
```
